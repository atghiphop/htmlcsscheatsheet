CSS Syntax

selector {
    property: value;
}

This is how you write CSS rules.

Selectors

Selector Description Example
* Universal selector 
* {
    margin: 0;
    padding: 0;
}
element Type selector 
p {
    color: blue;
}
.class Class selector 
.highlight {
    background-color: yellow;
}
#id ID selector 
#main {
    width: 100%;
}
[attribute] Attribute selector 
input[type="text"] {
    border: 1px solid black;
}
